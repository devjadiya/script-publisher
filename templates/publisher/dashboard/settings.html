About WikiScriptSync

WikiScriptSync is a Toolforge application built as part of the Developer Skill Development Program India 2025, implementing the Community Wishlist Survey 2022 proposal.

The tool uses OAuth only for identity verification. All wiki edits are made either via the user's own BotPassword or a pre-filled edit URL the user submits themselves. The notification bot posts talk-page messages to alert users of upstream changes — it never auto-publishes.

Source code: gitlab.wikimedia.org/toolforge-repos/script-publisher
