{# How-to steps #}
How WikiScriptSync works
1. Fetch repo

Enter a GitHub or GitLab URL. JS and CSS files appear instantly.

2. Map files

Set the destination wiki page for each file, e.g. User:Dev_Jadiya/script.js.

3. Review diff & publish

See the diff, get an edit URL, copy the content, paste it on the wiki yourself.

4. Get notified of updates

When GitHub changes, WikiScriptSyncBot posts on your meta talk page.
