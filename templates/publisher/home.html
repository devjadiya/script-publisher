WikiScriptSync

Synchronize JavaScript and CSS gadgets from public Git repositories to Wikimedia wikis — with diff review, audit logging, and change notifications.

Get started Wishlist proposal

1. Connect repository

Enter a public GitHub or GitLab URL. Files are fetched directly via the platform API — no third-party services.

2. Map to wiki pages

Map each file to a destination page such as User:You/script.js or MediaWiki:Gadget-Name.js.

3. Get notified of changes

When the source repository changes, you receive a talk-page notification so you can review and decide whether to update.

You are always in control

WikiScriptSync never edits wiki pages automatically. The notification flow alerts you to upstream changes and lets you review the diff before deciding to publish. Publishing is done either via your own BotPassword or a pre-filled edit URL you open yourself.
